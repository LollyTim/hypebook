# Welcome to HypeBook

Your AI-powered, open-source guide to building on Hyperliquid's HyperEVM.

Follow these steps to start building in minutes:

## What is Hyperliquid?

Hyperliquid is a high-performance blockchain platform that combines the best of both worlds:

- **HyperCore**: Lightning-fast spot and perpetual markets
- **HyperEVM**: Full Ethereum compatibility for smart contracts

This unique dual-layer architecture gives you the speed of centralized exchanges with the security and programmability of decentralized platforms.

### Key Features:

- **HYPE Token**: Native gas token on HyperEVM with 18 decimals
- **EIP-1559 Support**: Base fees are burned, priority fees are also burned (unlike most EVM chains)
- **Cancun Hardfork**: Uses Cancun without blobs
- **HyperBFT Consensus**: Inherits security from HyperBFT consensus
- **No WebSocket Support**: Currently no websocket JSON-RPC support on main RPC endpoint

## Quick Start Guide

### Step 1: Set Up Your Wallet

Install a Web3 wallet (like MetaMask) and add the HyperEVM network:

**Network Details:**

- **Network Name**: HyperEVM
- **RPC URL**: `https://rpc.hyperliquid.xyz/evm`
- **Chain ID**: 999
- **Currency Symbol**: HYPE
- **Block Explorer**: `https://explorer.hyperliquid.xyz`

[View Wallet Setup Guide ‚Üí](/docs/installation)

### Step 2: Get Testnet Funds

Use the official Hyperliquid faucet to request testnet tokens:

**Testnet Details:**

- **RPC URL**: `https://rpc.hyperliquid-testnet.xyz/evm`
- **Chain ID**: 998
- **Faucet**: [Hyperliquid Testnet Faucet](https://faucet.hyperliquid.xyz)

[Get Testnet Tokens ‚Üí](/docs/testnet/faucet)

### Step 3: Install Dependencies

```bash
npm install --save ethers
# or
yarn add ethers
```

For development frameworks:

```bash
# Using Hardhat
npm install --save-dev hardhat @nomicfoundation/hardhat-toolbox

# Using Foundry
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

### Step 4: Deploy Your First Contract

Create a simple smart contract:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.19;

contract HelloHyperEVM {
    string public message = "Hello from HyperEVM!";

    function setMessage(string memory newMessage) public {
        message = newMessage;
    }

    function getMessage() public view returns (string memory) {
        return message;
    }
}
```

Deploy using your preferred framework:

**Hardhat:**

```bash
npx hardhat compile
npx hardhat run scripts/deploy.js --network hyperevm
```

**Foundry:**

```bash
forge build
forge create HelloHyperEVM --rpc-url https://rpc.hyperliquid.xyz/evm
```

[Deploy Your First Contract ‚Üí](/docs/examples/contracts)

### Step 5: Transfer HYPE to HyperEVM

To move HYPE from HyperCore to HyperEVM, send HYPE to the bridge address:

```solidity
// Bridge address for HyperCore to HyperEVM transfers
address constant BRIDGE_ADDRESS = 0x2222222222222222222222222222222222222222;
```

### Step 6: Interact with HyperCore

Use precompiled contracts to interact with HyperCore markets:

```solidity
// Read from HyperCore order books
interface IHyperCore {
    function getOrderBook(uint32 marketId) external view returns (bytes memory);
    function getBalance(address user, uint32 marketId) external view returns (uint256);
}
```
